# Project Title

1. Hero Section (Top):

Slogan: "Plant a Tree, Grow a Future."

Mission: The text mentions a goal to plant 1 million trees to make the Earth greener.

Visuals: It uses a light green background with realistic palm leaves to create an eco-friendly aesthetic.

CTA: There is a "Get Involved" button.

2. Product Selection (Center - "Choose Your Trees"):

This section functions as an e-commerce interface where users can select trees to buy or sponsor.

Three product cards are visible: Mango Tree (৳500), Guava Tree (৳350), and Jackfruit Tree (৳800).

Each card includes an image, a short description, price in Bangladeshi Taka (৳), and an "Add to Cart" button.

3. Categories (Left Sidebar):

A filter menu allows users to browse different types of plants, such as Fruit Trees, Medicinal Trees, Timber Trees, and Evergreen Trees.

4. Shopping Cart (Right Sidebar):

There is a "Your Cart" section on the right to track selected items and the total cost (currently showing 0).
